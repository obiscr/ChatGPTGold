# ChatGPTGold

ChatGPT Gold增强了 [ChatGPT](https://plugins.jetbrains.com/plugin/20603-chatgpt) 插件的功能，提供了强大对话管理功能（比OpenAI官方的功能还要强）。对于ChatGPT和GPT-3.5-Turbo，提供了：
+ 会话列表
+ 删除对话
+ 修改对话标题
+ 查看对话细节
+ 导出对话
+ 支持历史对话的上下文，你可以打开任何对话历史并继续对话
+ 其他...

在未来，我计划增加更多的功能。
